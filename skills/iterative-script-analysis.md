# iterative-script-analysis

Iteratively run scripts and read output files to analyze or investigate a system, dataset, or codebase

## Steps

1. Execute a script to generate data or perform an initial operation
2. Read the resulting output files for analysis
3. Read additional related files for context
4. Execute a follow-up script based on findings
5. Read further output files to examine results
6. Read more supporting files
7. Execute a final script to conclude the investigation or apply fixes
