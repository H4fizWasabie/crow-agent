# Deep Research Protocol

## Steps

### 1. Scope
Understand what exactly I need to know. If the request is vague, ask ONE clarifying question — then proceed.

### 2. First Pass — Broad Search
- Call `web_search` with 2-3 different queries covering different angles
- Pick the 3-5 most promising results and `web_fetch` each
- Note what's covered and what's missing

### 3. Second Pass — Fill Gaps
- Identify specific gaps or claims that need verification
- Search again with targeted queries based on what you found in step 2
- Fetch 2-3 more specific sources

### 4. Third Pass — Cross-Reference
- Check for contradictions across sources
- Search for any remaining unknowns
- Verify key claims with at least 2 independent sources where possible

### 5. Synthesize
Structure the output as:
- **Summary** — 2-3 sentence bottom line
- **Key Findings** — bullet points with source context
- **Details** — deeper dive organized by theme
- **Sources** — numbered links to all URLs consulted

### 6. Deliver
Send the full report. Then offer: *"Want me to go deeper on any specific aspect?"*
