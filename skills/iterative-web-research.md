# iterative-web-research

Conduct iterative web research by searching and fetching multiple sources, scraping content, and processing data with scripts.

## Steps

1. Search the web for initial information
2. Fetch and read web pages
3. Refine search and fetch more sources
4. Scrape specific page content
5. Run scripts to process collected data
