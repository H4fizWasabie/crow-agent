# procurement-rfq-generation

Retrieve and process procurement data through queries and command executions, read files, and draft a Request for Quotation.

## Steps

1. Query procurement data for initial information
2. Execute commands to process or analyze data
3. Read files for additional details
4. Run further commands as needed
5. Re-query procurement data for refinement
6. Draft an RFQ document based on the analysis
